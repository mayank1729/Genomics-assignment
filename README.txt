Each folder corresponding to a drug contains the model with the best prediction on the leaderboard, the relevant data required to produce the output in the format specified in the competition and the output file in the format specified in the competition.
Folder Group_3_backup has a .zip file with all the drug folders for the purpose of backup.
Execute the jupyter notebooks in each folder to produce the output files.

NOTE: You will need to install additional python libraries to run the code which include xgboost, pandas, numpy, sklearn, matplotlib and tensorflow.
